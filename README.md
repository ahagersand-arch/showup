# ShowUp – Privat visning

Premium “glassmorphism”-demo byggd med Next.js 14, TypeScript och Tailwind.

## Så kör du lokalt

```bash
npm install
npm run dev
```

Öppna [http://localhost:3000](http://localhost:3000).
